# palindromic

**Install Node.js and NPM**<br />

https://nodejs.org/en/download/<br />
npm install -g npm<br /><br />

- **Check version**<br />
node -v<br />
npm -v<br /><br />

- **Install express**<br />
npm init<br />
npm install express --no-save<br /><br />

- **Build and run the project**<br />
npm run build<br />
npm start<br />
npm run dev<br />
